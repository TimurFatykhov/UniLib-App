# UniLib
UniLib - your personal librarian

## Usage

To start use application:



```bash
$ git clone https://github.com/TimurFatykhov/UniLib.git
```

The next step is to open our project and run it at Xcode 9.0+
```bash
$ open -a Xcode a.xcodeproj
```
